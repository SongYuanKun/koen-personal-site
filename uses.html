Uses

我日常用来写代码、做内容和维护服务的工具栈。

每个工具都讲清楚"解决什么问题"，而不是只列名字。偏向轻量、低维护成本、真实解决问题的选择。

编辑器 & IDE
主力
IntelliJ IDEA
Java 后端主力 IDE。用于 Spring Boot 项目开发、远程调试和大规模重构，代码分析和跳转能力无可替代。
日常
VS Code
前端、Markdown、脚本和内容仓库维护。插件生态灵活，启动快，适合轻量编辑和跨语言场景。
AI 辅助
Claude Code CLI
终端内 AI 编程助手，用于代码生成、重构建议、文档整理和多 Agent 任务编排，深度整合到日常工作流。
AI 工具
对话 & 推理
Claude / GPT
复杂推理、代码审查和长文档分析的主力 AI。Claude 在结构化输出和源码分析场景表现突出。
内容创作
Qwen / GLM
中文内容创作和本地私有部署场景。国产模型对中文语境理解更准确，成本控制上有明显优势。
图像
Midjourney / DALL·E
封面图和配图生成。文章封面遵循品牌暗色规范（#0f172a 背景 + Indigo 主色）统一视觉语言。
部署 & 运维
服务管理
1Panel + OpenResty
个人站点和服务入口的统一管理面板。配合 Docker 容器化部署，降低运维复杂度，支持通配证书和反向代理配置。
容器化
Docker / Compose
所有自建服务优先容器化，环境隔离、升级回滚和迁移成本显著降低。TrendRadar 等项目完全基于 Compose 编排。
云服务
阿里云 ECS
主力服务器。搭配 ufw 防火墙、frpc 内网穿透和 1Panel 管理面板构成完整的个人云基础设施。
设计 & 内容
视觉设计
Figma
品牌视觉规范整理和封面设计。用于维护设计令牌（颜色、字体、间距）和输出图文卡片模板。
快速制图
Canva
小红书图文和公众号封面的快速输出工具。Figma 用于设计规范，Canva 用于批量生产执行。
API 调试
Postman / Apifox
后端接口调试和文档维护。Apifox 在中文文档和团队协作场景体验更好，个人项目两者都用。
效率 & 版本管理
版本控制
Git / GitHub
代码和内容都进仓库，便于 diff、回滚和跨设备同步。内容仓库也用 Git 管理，让知识沉淀可追溯。
终端
Linux Shell / tmux
服务器巡检、部署、日志排查和自动化脚本的基础入口。tmux 用于长时间运行任务的会话保持。
自动化
GitHub Actions
CI/CD 和内容发布自动化。用于代码检查、静态站构建和定时任务触发，降低手动操作成本。
静态托管
GitHub Pages
免费静态站托管。个人主站在 songyuankun.github.io 有 GitHub Pages 镜像，作为自建服务器的备用访问入口。
